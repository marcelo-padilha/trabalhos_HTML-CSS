Ementa do Curso: HTML e CSS

Objetivo do Curso

Capacitar os alunos do Ensino Médio a compreender e desenvolver a estrutura e o estilo de páginas web modernas, criando projetos interativos e responsivos que servem de introdução ao mundo da programação e do design digital.

Conteúdo Programático

  • Fundamentos da Web: Como a internet funciona, clientes, servidores e hospedagem.
  • HTML5 Semântico: Tags de texto, imagens, links, listas, tabelas e estruturação de layout (header, main, footer, section).
  • Formulários HTML: Inputs, botões e validações básicas.
  • CSS3 - Estilização Visual: Cores, tipografia, bordas e backgrounds.
  • CSS Layouts: Modelo de caixas (Box Model), Flexbox e CSS Grid.
  • Versionamento e Publicação: Uso básico do GitHub e publicação via GitHub Pages.
